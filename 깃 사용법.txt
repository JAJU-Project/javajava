깃 사용법

1) 상단메뉴 - 터미널(Terminal) - 새터미널(New Terminal)

2) 왼쪽 메뉴 돋보기 아래에 아이콘 눌러서 코드 변경사항 확인

3) github의 Repository(저장소) 주소 입력
    git remote add origin https://github.com/JAJU-Project/javajava.git 입력

4) git remote -v github의 Repository 주소를 확인으로 확인
    origin https://github.com/JAJU-Project/javajava.git (fetch)
    origin  https://github.com/JAJU-Project/javajava.git (push)
    나오면 정상

5) 변경된 파일 중 commit 할거 선택하기 + 버튼
    그럼 "스테이징된 변경사항"에 해당 파일 추가됨
    - 버튼 누르면 commit 대상에서 제외됨

6) "스테이징된 변경사항" 위에 체크(V) 버튼 누르면 
    간단한 변경사항 메모 할 수 있는게 뜸 > 변경사항 작성 후 엔터

7) git push origin main
    main <- 이 부분은 브랜치 명
